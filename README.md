# 彩票走势分析站

多彩种彩票开奖查询与走势统计分析 Web 应用（演示数据，非官方开奖源）。

## 功能

### 支持彩种

| 彩种 | 开奖查询 | 走势分析 | 数据统计 | 选号工具 |
|------|----------|----------|----------|----------|
| 双色球 | ✅ | ✅ 红/蓝球矩阵 + 折线图 | ✅ 冷热号 | ✅ |
| 大乐透 | ✅ | ✅ 前区/后区走势 | ✅ | ✅ 和值过滤 |
| 福彩3D | ✅ | ✅ 按位走势 | ✅ | ✅ 机选 |
| 排列三 | ✅ | ✅ 按位走势 | ✅ | ✅ 机选 |

### 模块说明

- **开奖查询**：最新一期展示、历史列表、期号筛选、分页条数
- **走势分析**（双色球重点）：红球 01–33、蓝球 01–16 遗漏走势表；和值/跨度/奇数折线图
- **数据统计**：近 N 期和值、号码出现频次、热号/冷号标签
- **选号工具**：机选 1/5 注、复制、和值范围过滤（双色球/大乐透）

## 使用

1. 双击打开 `index.html`，或使用本地服务器：

```bash
# Python
python -m http.server 8080

# Node
npx serve .
```

2. 浏览器访问页面，顶部切换彩种，下方切换功能 Tab。

## 更新数据（最近 30 期，真实开奖）

项目自带抓取脚本，从 [500.com 开奖公告](https://datachart.500.com) 拉取各彩种最近 30 期并写入 `data/lottery-data.js`：

```bash
node scripts/fetch-lottery.js
```

需联网。抓取后刷新页面即可。数据仅供参考，请以福彩/体彩官方公布为准。

当前样本最新一期示例（双色球）：**2026055 期 · 2026-05-17 · 红 04 11 24 25 32 33 · 蓝 13**
